u14javpls
===============

Shippable CI image for Java on Ubuntu 14.04. Available jdk versions:

1. openjdk6
2. openjdk7
3. oraclejdk7
4. oraclejdk8

## Services:

1. elasticsearch 1.5
2. memcached 1.4
3. mongodb 3.0
4. mysql 5.6
5. postgres 9.4
6. rabbitmq 3.5
7. redis 3.0
8. selenium 2.45
9. sqllite 3
